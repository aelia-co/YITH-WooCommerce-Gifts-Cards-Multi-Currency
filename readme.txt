Allow your users to purchase and give gift cards, an easy and direct way to encourage new sales.

== Description ==

Gifts are always welcome and YITH WooCommerce Gift Cards lets you offer your users a powerful sales tool able to enhance the purchases made on your site. Allow your users to give discount coupons and gift cards of the amounts you want, and encourage your customers to gift them to possible new users of your shop. In this way, you will kill two birds with one stone: the possibility to make appreciated presents, and the chance to promote your products thanks to convenient coupons.

= Main features =

* Create new products: Gift Cards.
* Set the amount you want to use for each Gift Card.
* Send via email the Gift Card code once the order has been completed.
* Use the Gift Card until the depletion of the amount.

Discover all the features of YITH WooCommerce Gift Cards and install it in your theme: the result will be extremely satisfying.

== Installation ==

Important: firstly, you have to download and activate the WooCommerce plugin, which is mandatory for YITH WooCommerce Gift Cards to be working. If the WooCommerce plugin is active, you will just need to activate the YITH WooCommerce Gift Cards plugin.

= Configuration =

YITH WooCommerce Gift Cards will add a new tab called "Gift Cards" inside the YIT Plugins menu item. There you are able to configure all plugin settings.

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH WooCommerce Gift Cards` from Plugins page.
